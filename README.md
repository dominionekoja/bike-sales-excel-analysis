#  Bike Sales Analysis (Excel Dashboard)

##  Project Overview
This project analyzes bike sales data using Microsoft Excel. The goal is to uncover customer purchasing patterns and sales trends using pivot tables and an interactive dashboard.

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning Techniques

##  Key Insights
- Sales distribution by gender and marital status
- Impact of income on bike purchases
- Sales trends by region and age group
- Customer segmentation patterns

##  Dashboard Preview
https://github.com/dominionekoja/bike-sales-excel-analysis/blob/main/dashboard_preview.png


##  How to Use
1. Download the Excel file from the `dashboard` folder
2. Open in Microsoft Excel (Desktop version recommended)
3. Use slicers to interact with the dashboard
